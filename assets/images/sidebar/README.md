# Sidebar image requests

Place sidebar-specific icons in the theme folders:

- `light/`: icons for light mode.
- `dark/`: icons for dark mode.

Each icon can be either `.png` or `.svg`. The site tries `.png` first and automatically falls back to `.svg` if the PNG is missing.

Recommended icon colors:

- Light theme primary: `#061724`
- Light theme accent: `#0ea5ff`
- Light theme muted strokes: `#516b7d`
- Dark theme primary: `#f0fbff`
- Dark theme accent: `#69d5ff`
- Dark theme muted strokes: `#98bdd2`

Recommended filenames:

- `brand-mark.png` or `brand-mark.svg`
- `menu.png` or `menu.svg`
- `home.png` or `home.svg`
- `socials.png` or `socials.svg`
- `dropdown-arrow.png` or `dropdown-arrow.svg`
- `privacy-choices.png` or `privacy-choices.svg`
- `tracking-choice.png` or `tracking-choice.svg`
- `tracking-off.png` or `tracking-off.svg`
- `privacy.png` or `privacy.svg`
- `terms.png` or `terms.svg`
- `light.png` or `light.svg`
- `dark.png` or `dark.svg`
- `external-warning.png` or `external-warning.svg`
- `analytics.png` or `analytics.svg`
